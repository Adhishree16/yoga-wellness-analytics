# yoga-wellness-analytics
This project analyzes yoga practice data to understand its impact on stress, sleep quality, mood, and overall wellness using SQL and Power BI.

## 📌 Overview
This project is a data analytics case study focused on understanding how yoga practices impact physical and mental well-being. It uses a realistic dataset to analyze stress levels, sleep quality, mood scores, and other health indicators.

The project combines:
- SQL for data analysis
- Power BI for interactive dashboards
- Excel for preprocessing

It demonstrates how wellness activities like yoga can be quantified using data-driven insights.

---

## 🎯 Problem Statement
In today’s fast-paced lifestyle, individuals face increasing stress, poor sleep quality, and reduced mental well-being.

This project aims to:
- Analyze the effect of yoga on stress reduction
- Measure improvement in sleep quality and mood
- Identify the most effective yoga types for wellness
- Understand patterns across age, gender, and lifestyle factors

---

## 📊 Dataset Description
A synthetic but realistic dataset containing **1000+ records** was used for analysis.

### 🧾 Features:
- User ID
- Age
- Gender
- BMI
- Yoga Type (Asana / Pranayama / Meditation)
- Duration of practice (minutes)
- Frequency (Daily / Weekly)
- Stress Level Before Yoga
- Stress Level After Yoga
- Sleep Quality Score
- Mood Score
- Heart Rate
- Calories Burned

---

## 🛠️ Tools & Technologies
- 🗄️ SQL – Data cleaning, transformation, and analysis
- 📊 Power BI – Dashboard creation and visualization

---

## 🧠 SQL Analysis Performed
The following analytical queries were performed:

- Average stress reduction by yoga type
- Sleep quality improvement analysis
- Mood score variation across demographics
- Correlation between frequency and wellness improvement
- Calories burned vs duration analysis
- BMI vs yoga effectiveness comparison

## 📈 Dashboards Overview

### 1️⃣ Stress Reduction Dashboard
- Comparison of stress levels before and after yoga
- Yoga type effectiveness in stress management
- Trend analysis across users

### 2️⃣ Sleep & Mood Analysis Dashboard
- Sleep quality improvement after yoga practice
- Mood score variations by yoga type and age group
- Emotional wellness patterns

### 3️⃣ Overall Wellness Dashboard
- Heart rate improvement trends
- BMI vs wellness correlation
- Age-wise performance analysis
- Yoga frequency impact on health

---

### 📌 Key Insights
- Regular yoga practice significantly reduces stress levels
- Pranayama shows the highest improvement in sleep quality
- Consistent daily practice is more effective than occasional sessions
- Younger age groups show faster wellness improvement trends
- Yoga positively impacts mood stability and emotional balance
